Start writing a minimal selection for the decay J/psi -> MuMu.
In this first exercise we will just loop over the muons.

  - Put all your algorithms in a GaudiSequencer
  - Write a DVAlgorithm "TutorialAlgorithm" 
    - Use Standard loose muons as input
    - Make sure you have the doxygen documentation of DVAlgorithm
    - Use emacs to create the .cpp and .h files and tell it to make 
      a DVAlgorithm
    - Get the Particles from the desktop
    - Loop over them and plot the Pt and the momentum.
    - Get the primary vertices and use the geometry tool to get the
      impact parameter of your muons.
  - Use some J/psi signal data to run on

General tips:
    - Make sure all your global variables are initialised in the constructor
    - Avoid hard-coding as much as possible. All cuts should be 
      properties of your algorithm.
    - Avoid duplication of code. Make methods when necessary.

The Tutorial package already provides a complete requirements file.

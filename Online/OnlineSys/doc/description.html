The ONLINE project contains the infrastructure software for all data processing applications in the LHCb online system - mainly the high level trigger application (HLT) which can be found in the HLT web pages

Reporting bugs

Please report bugs by E-mail to the Online Group (M.Frank, N.Neufeld, E.v.Herwijnen).

    Version built against LHCb v25r0. Fix to important memory leak in gaucho and new version of presenter including MonObjects. Bug fixes in RawBankSizes. Please note the important note from Eric about additional job options. The following change to your options files is required: ApplicationMgr.TopAlg ={"UpdateAndReset"}; (UpdateAndReset needs to be the first algorithm in the TopAlg list).
